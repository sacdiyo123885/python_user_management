# Python User Management System

This is a simple Python developer project demonstrating basic CRUD operations using Python and JSON file handling.

## Features
- Add users (name & email)
- View all users
- Delete a user by email
- Exit application safely

## Technologies Used
- Python (core concepts, OOP, file handling)
- JSON for simple data storage

## Purpose
This project showcases Python development skills including:
- Clean and reusable code
- File-based data management
- CRUD operations logic
